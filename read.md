# Site de Divulgação - Personal Trainer

Este projeto foi desenvolvido como parte do Projeto de Extensão (PEX), com foco na criação de um site pessoal para um(a) Personal Trainer. O objetivo é promover seus serviços de forma profissional, responsiva e integrada às redes sociais.

## Tecnologias utilizadas
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Estrutura Inicial
- Página inicial com layout responsivo
- Sessão de apresentação (hero section)
- Integração futura com API do Instagram
- Rodapé institucional
